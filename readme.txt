How to use

1) Open generateData.m in MALTAB
    - Generates data for 30 sec
	- y = sin(x) for x = 0:0.01:30
    - Writes data to a text file (over 30 sec)

2) Run gnuplot
    - Go to terminal and type "gnuplot" (Press Enter)
    - Then type "load 'realtime.p'"

3) Watch plot animation
    - To close it, do the following
	1) Press control
	2) Click on gnuplot terminal
	3) IMMEDIATELY press "C"
	4) "Ctrl + C" will stop it but you need to be fast
